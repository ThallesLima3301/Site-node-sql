Relatório ----------------------------------------------------------

Instale no Terminal ANTES de rodar o código!
//Instalar o Node modules
npm install express -- save

//Instalar o SQLite
npm install sqlite3

//Rodar o site com o comando abaixo:
npm run dev

O programa inicia na Página Principal, vai ter várias notícias 
relacionadas ao assunto do site e lá você terá a escolha de ir 
para o Atendimento e o Sobre Nós;

Dentro do Atendimento, caso não tenha feito o Cadastro terá um ícone
que ao clicar nele, será redirecionado para a página para
fazer o Cadastro;

No Cadastro pediremos para o usuário escrever suas informações 
pessoais para armazenarmos seus dados no nosso Banco de Dados.

Após cadastrar o usuário poderá fazer o Login clicando no mesmo na
aba do Atendimento;

Na área do Login precisa inserir o seu e-mail e senha de Cadastro;

Depois de logado, o usuário será tranferido para a Página do Usuário,
onde poderá escolher entre Agendar Consulta e Visualizar as Consultas
Cadastradas.
